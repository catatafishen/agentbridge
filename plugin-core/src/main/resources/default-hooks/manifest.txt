# AgentBridge default hook files — one entry per line.
# Hooks are written in JavaScript and run in-process via the embedded Rhino engine
# (see JsHookEngine), so a single .js file works on every OS and JetBrains IDE — no
# paired .sh/.ps1 variants and no external shell, PowerShell, or Node runtime.
#
# The *.json config files map an MCP tool to the scripts that run for it. They are bundled
# as plain, editable examples: copy or edit them under <project>/.agentbridge/hooks/ to
# extend or override the default behaviour. They are NOT generated at runtime.
